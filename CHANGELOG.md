# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-14

### Added

- Excalidraw-based canvas editor
- CodeMirror 6 code editor with vim mode, syntax highlighting, and autosave
- Built-in terminal powered by Go PTY backend
- Git-based version control with sync, history, and revert at file, folder, and project levels
- Rich text editor using TipTap
- File management with create, rename, and delete operations
- Live activity panel
- Environment validation and setup wizard
- Go gRPC engine with health, workspace, terminal, generate, and event services
- Electron desktop shell with IPC bridge and bearer token auth
- Toast notifications, progress indicators, and onboarding flow
- Engine crash recovery
