Everything you need in one workspace

Visual Canvas

Draw wireframes, user flows, and architecture diagrams on an Excalidraw-powered canvas. Your designs live alongside your code.

AI Chat

Chat with an AI that can see your files, your designs, and your terminal. Ask it to write code, refactor, or explain — every change is tracked.

Project Sections

Structured rich text for your app's purpose, tech stack, brand guidelines, and user stories. Feeds into AI so it understands what you're building.

Code Editor

CodeMirror 6 with syntax highlighting, vim mode, and autosave. A real editor for real work.

Built-in Terminal

Full PTY terminal powered by a Go backend. Run your dev server, tests, and scripts without leaving the app.

Source Control

Git staging, commits, tagged history, and one-click revert. AI actions are auto-committed so you always know what changed and why.

How it works

Describe your app visually and in writing. Let AI turn it into code. Review, iterate, ship.

1. DescribeCanvas + rich text sections
→
2. GenerateAI writes tracked code
→
3. ReviewDiff, revert, or re-run
→
4. ShipRun in the built-in terminal
